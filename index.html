Day 1 • HTML/CSS • Git

Αλλάζω καριέρα σε Full-Stack Dev.

Χτίζω από το μηδέν, κάθε μέρα, με πραγματικά deliverables. Σήμερα: landing page + repo με σωστά commits.

Δες το πλάνο Στείλε μήνυμα
3–4hσήμερα
12εβδομάδες
3projects

Deliverable

  1. Responsive landing (mobile → desktop)
  2. Repo στο GitHub με 3+ commits
  3. (Bonus) GitHub Pages live link

Tip: κάνε μικρά commits, όχι “ένα και τέλος”.

Τι θα καλύψουμε συνολικά

Frontend

HTML/CSS/JS → React, routing, forms, patterns.

Backend

Node/Express → APIs, auth, validation, error handling.

Database

CRUD, σχέσεις, migrations + deployment.

Για μένα

Ξεκινάω σήμερα, με συνέπεια. Στόχος: portfolio που “μιλάει” και αλλαγή καριέρας.

  • Καθημερινό session (23:00)
  • Ασκήσεις + review από εμένα
  • Project-based μάθηση

Κανόνας

Κάθε μέρα κάτι να “φαίνεται”: commit, feature, demo.

Επικοινωνία

Απόψε, μόλις το ανεβάσεις, στείλε μου το repo link και θα στο κάνω review.
